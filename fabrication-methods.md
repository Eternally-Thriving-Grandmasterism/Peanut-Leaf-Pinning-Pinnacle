# Peanut Leaf Pinning Fabrication Methods Lattice

Recommended Scalable Path: Dense Multi-Level Nanoimprint Lithography (NIL) + Wrinkle Induction

1. Master Mold Preparation
   - Si/Ni shim: e-beam/LIL for dense 5–12 μm papillae + 50–400 nm wrinkle overlay

2. Resin Coating & Imprint
   - R2R UV resin on PET/PC → sequential NIL (micro papillae dense pack → nano wrinkles)

3. Demold & Surface Modification
   - Separate → optional plasma + hydrophobic coat (fluorosilane / wax) for pinning boost

4. Wrinkle Enhancement
   - Thermal/strain-release buckling for nanoscale folds on pillars

Alternative Paths:
- Colloidal + plasma wrinkling (spheres for dense micro → buckling for nano)
- Femtosecond laser + chemical texturing (direct dense roughening)

Valence Check:
- Dry/low-waste processes preferred
- Recyclable masters
- Passive ultra-pinning → eternal mercy
